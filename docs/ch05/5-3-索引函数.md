# 5-3 索引函数

本节提及的索引函数并不是指用来进行索引的函数，而是指返回值为索引的函数，它们包括nonzero()、argmax()和argmin()。其中，nonzero()返回非零数的索引，argmax()，argmin()分别返回最大和最小数的索引。

```{code-cell} ipython3
a = np.array([0,-5,0,1,3,-1])
np.nonzero(a)
```


```{code-cell} ipython3
a.argmax()
```


```{code-cell} ipython3
a.argmin()
```
