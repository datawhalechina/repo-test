# 2-1 变形的两种方式

数组的变形现象主要来自于2个方面，其一是由数组中元素组织方式的变化而导致的变形，其二是多个数组合并或单个数组拆分时产生的维度变化导致的变形。因此，关于数组维度变化的研究是数组变形操作的核心内容，尤其是在高维数组中，熟练掌握变形函数有利于使得数据处理操作更为便捷高效。
