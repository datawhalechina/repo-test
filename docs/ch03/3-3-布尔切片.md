# 3-3 布尔切片

在实际使用中，利用布尔数组进行切片也是一种常见方式，使用方法为：若要保留某一个维度的若干维数，则只需传入相应位置为True且其余位置为False的布尔数组。例如，保留外层维度中维数为0和2的子数组：

```{code-cell} ipython3
target[[True, False, True, False], :, :]
```
