# 3-2 逐元素索引

如果[]符号对应维度的输入值都是长度相同的列表（或一维numpy数组）时，其结果返回的并不是子数组。此时输入值分别代表元素在各个维度的索引值，例如下面的例子等价于取出target[0, 0, 0]和target[1, 1, 1]而不是一个大小为$2\times 2\times 2$的子数组：

```{code-cell} ipython3
target[[0, 1], [0, 1], [0, 1]] # 单个元素逐个取出
```

```{code-cell} ipython3
target[0:2, 0:2, 0:2] # 大小为2*2*2的子数组
```
